TicTacToe Project

I developed a fully functional Tic-Tac-Toe game in Java, implementing both the back-end logic and front-end interface. The back-end handles game mechanics, including player turns, win conditions, and validation for moves. The front-end provides a clean and interactive graphical user interface, ensuring an engaging user experience. The project emphasizes object-oriented programming principles, with modular code for easy scalability and maintenance. It was designed as a standalone application and demonstrates my ability to integrate logic and interface seamlessly. This project highlights my skills in Java development, problem-solving, and creating interactive applications.

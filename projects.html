CITAP Website

I created a modern and responsive website for Lethbridge Polytechnic using Wix, focusing on usability and accessibility. The site features a dynamic home page, a streamlined navigation system, and dedicated sections for academic programs, events, and admissions. An integrated events calendar ensures the community stays informed about upcoming activities. The admissions section provides clear, step-by-step guidance for prospective students. The design is user-friendly, visually appealing, and optimized for both desktop and mobile devices. This project showcases my skills in web design, content organization, and creating functional, user-centric digital platforms.
